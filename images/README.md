# Images

Bu klasor, `yeni.md` icinde gercekten kullanilacak secilmis gorseller icindir.

Kullanim mantigi:
- `foto/` klasoru arsiv olarak kalabilir.
- `images/` klasorune yalnizca Markdown icinde kullanacagin gorselleri koy.
- Dosya isimlerini kisa ve anlamli tut.

Ornek adlar:
- `fig_01_block_diagram.png`
- `fig_02_vout_transient.png`
- `fig_03_input_filter_bode.png`

Markdown ornegi:

```md
![Input filter bode plot](images/fig_03_input_filter_bode.png)
```


Ek klasor:
- `odt_embedded/`: `yeni.odt` icinden cikartilan ve ilk asamada kullanilacak gorseller
